## Diagrammes UML (Mermaid)

Le dossier `docs/` contient les différents diagrammes d'architecture du projet, modélisés avec la syntaxe *Mermaid.js*.
Pour les visualiser, vous avez trois options :
1. **GitHub** : Naviguez dans votre répertoire GitHub. Les fichiers Markdown (`.md`) contenant des blocs `mermaid` sont automatiquement affichés sous forme de diagrammes graphiques dans l'interface de GitHub.
2. **VS Code / IntelliJ IDEA** : Installez une extension comme "Mermaid Preview" ou "Markdown Preview Mermaid Support" pour les visualiser en local pendant l'édition.
3. **Outil en Ligne** : Copiez le bloc de code (commençant par `graph TD`, `usecaseDiagram` ou `classDiagram`) dans l'éditeur officiel [Mermaid Live Editor](https://mermaid.live).
